## 📌 Project Overview
An autonomous, AI-driven diagnostic system designed to detect COVID-19 and other lung conditions (Viral Pneumonia, Lung Opacity, and Normal) from patient X-ray and CT scan images. This project was developed as my **Computer Science Graduation Project**, where I served as the **Team Leader**, successfully achieving an **A+ grade**.

The project consists of two main components:
1. **Deep Learning Model:** A robust Convolutional Neural Network (CNN) trained on a massive dataset of medical images.
2. **Mobile Application:** A cross-platform mobile app built with Flutter, allowing seamless image uploads and real-time diagnostic reporting via Firebase.

## 🚀 Key Features
* **Multi-Class Classification:** Accurately distinguishes between COVID-19, Viral Pneumonia, Lung Opacity, and healthy lungs.
* **High Accuracy:** Reached **96.04% accuracy** on CT scan analysis.
* **Real-time Mobile Interface:** Integrated the AI model with a Flutter app for end-user accessibility.
* **Cloud Integration:** Utilized Firebase for fast and secure data handling.

## 🛠️ Technology Stack
* **Machine Learning / AI:** Python, TensorFlow, Keras, OpenCV, Google Colab.
* **Mobile Development:** Flutter, Dart.
* **Backend & Database:** Firebase.
* **Data Processing:** Pandas, NumPy, Matplotlib for visualization.

## 📊 Dataset Description
The model was trained on a comprehensive dataset curated from public medical repositories, ensuring high generalization ability:
* **X-ray Images:** 42,330 images.
* **CT Scans:** 14,486 images.
*(Note: Datasets were sourced from verified medical databases and Kaggle).*
